Mapping Tools

iD EDITOR

The iD editor is browser-based OpenStreetMap editor. It is fast and easy to use, and allows mapping from various data sources such as satellite and aerial imageries, GPS, Field Papers or Mapillary. The iD editor is a great way to make small and easy changes that do not require the advanced features of JOSM, more advanced mapping editor. (Source: Learnosm.org 2018)


JOSM EDITOR

Editing OpenStreetMap with JOSM is similar to editing with the iD editor. However, JOSM is a desktop application and it works a little bit differently. JOSM editor is a great tool for beginners who want to learn more. (Source: Learnosm.org 2018)
